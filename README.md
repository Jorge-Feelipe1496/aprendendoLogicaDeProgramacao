# 🧠 Aprendendo Lógica de Programação

Este repositório foi criado com o objetivo de reunir exercícios, exemplos e anotações sobre **lógica de programação**, servindo como base para quem está iniciando na área de desenvolvimento.

---

## 📚 Conteúdo

- Introdução à lógica de programação
- Estruturas de decisão (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do...while`)
- Operadores lógicos e aritméticos
- Funções e modularização
- Exercícios práticos

---

## 🚀 Objetivo

O objetivo é **praticar e consolidar conceitos fundamentais** de programação, que podem ser aplicados em qualquer linguagem.  
Aqui você encontrará exemplos simples e comentados para facilitar o aprendizado.

---

## 🛠️ Tecnologias Utilizadas

- Linguagem: **JavaScript** (pode ser adaptado para outras linguagens)
- Editor sugerido: **VS Code**
- Controle de versão: **Git & GitHub**
